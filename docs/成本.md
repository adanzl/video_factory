## 2. 成本结构总览

**日产 15 条**、默认 Z-Image（`prompt_extend=false`）、一次通过率 75%、含 **1.25×** 出图重制：

| 成本项 | 计费方式 | 日金额（15 条，估） | 月金额（×30，估） | 占比 |
| --- | --- | --- | --- | --- |
| Z-Image 文生图（7 张×15 条×1.25） | ~0.1 元/张 | **~13 元** | **~390 元** | **~55%** |
| DeepSeek V4-Flash | 按 Token | **~1.2～1.8 元** | **~36～54 元** | ~6% |
| CosyVoice **`cosyvoice-v3-flash`** | 按输入字符 | **~1.4～1.7 元** | **~42～51 元** | ~5% |
| Qwen2.5-VL-7B | 按次/Token | **~0.9～2.7 元** | **~27～81 元** | ~6% |
| FFmpeg / YOLO | 自建 | **0** | **0** | — |
| 可灵 std（**升级开启**，7 段全用） | 1.5 元/5s | **+~197 元** | **+~5910 元** | 可选叠加 |

**日现金合计（默认静图 · 无可灵）**：约 **17～22 元/日**（中性 **~19 元/日**）。  
**月现金合计（×30）**：约 **510～660 元/月**（中性 **~570 元/月**）。  
**若全开可灵 std 升级**：在上表基础上 **+~197 元/日** ≈ **+~5910 元/月**。

> 若改用万相或开启 `Z_IMAGE_PROMPT_EXTEND=true`（0.2 元/张），文生图费用约 **×2**。

### 单条成本与单镜秒数的关系

在 **口播字数固定（~1100 字）**、**封面不计 Z-Image** 的前提下，单条 API 成本可拆成 **固定项** 与 **随分镜数线性增长项**：

| 成本块 | 金额（估） | 是否随单镜秒数变 |
| --- | --- | --- |
| CosyVoice TTS | **~0.11 元/条** | 否（按总字数） |
| DeepSeek LLM | **~0.10 元/条** | 基本否（分镜/Prompt 略增，可忽略） |
| Qwen-VL 质检 | **~0.011 元/镜** | 是（每分镜 1 帧） |
| Z-Image 出图 | **0.125 元/镜**（0.1×1.25 重制系数） | 是 |
| **固定合计** | **~0.21 元/条** | — |
| **每增 1 镜** | **~+0.136 元/条** | — |

```text
正文秒数 T = 成片总秒数 − 1.5（片头）
分镜数 N = ⌈T ÷ 单镜秒数 S⌉
单条 API ≈ 0.21 + N × 0.136
单条画面 ≈ N × 0.125
```

**直觉**：正文时长固定时，**单镜秒数减半 → 分镜数约 ×2 → 单条成本约 +0.136×N（主要多出图）**。TTS 几乎不变。

## 5. 月度损益（创作激励）

本节回答：**月现金成本**（由日产量 ×30 折算）；以及 **全账号月播放要达到多少，激励才盖住现金**（未计人工、税费）。

### 盈亏平衡播放

B 站创作激励按播放量折算，此处用 [选题.md](./选题.md) 中性值：**每 1000 次播放 ≈ 2.3 元**。

```text
月总播放（次） ≈ 月现金成本（元） ÷ 2.3 × 1000
```

例：中性 **日产 15 条**、默认静图 → 月现金 **~570 元** → 570 ÷ 2.3 × 1000 ≈ **25 万次/月**（全账号播放加总）。

### 情景表（默认 Z-Image；括号内为 **全开可灵 std 升级** 的月成本）

| 情景 | 日产量 | 月现金成本（×30） | 单条成本 | 盈亏平衡（月总播放） |
| --- | --- | --- | --- | --- |
| 保守 | 10 条/日 | **~330～450 元**（~6.7k） | **~1.2～1.6 元**（~22） | **~14～20 万**（~290 万） |
| 中性 | **15 条/日** | **~510～660 元**（~6.7k） | **~1.2～1.6 元**（~15） | **~22～29 万**（~290 万） |
| 乐观 | 20 条/日 | **~660～870 元**（~6.7k） | **~1.2～1.6 元**（~12） | **~29～38 万**（~290 万） |

- **单条成本**：默认 **~1.2 元**；单条 **~520 次播放**即可接近覆盖制作费（与产量无关）。
- **盈亏平衡**：**日产 15 条** 时，全账号月播放约 **25 万** 可打平 API 现金（默认可灵关）。

### 单条播放够不够回本？

默认静图单条约 **1.2 元**。若某条播放 **3000 次**：

```text
该条激励收入 ≈ 3000 ÷ 1000 × 2.3 ≈ 6.9 元  >  1.2 元
```

单条 **~520 次**即可接近回本；**账号整体打平**仍看上文情景表**月总播放**（日产量越高，月总播放门槛越高）。

升级 **V4-Pro** 仅在三级回退终审时触发；日产 15 条时月增约 **数十～百 元** 量级，相对月总成本占比仍小。

## 6. 重制与失败成本

| 质检结果 | 额外 API | 额外画面（Z-Image 0.1 元/张） |
| --- | --- | --- |
| 一级通过 | 0 | 0 |
| 二级（重 2 张插画） | VL ×2，+~0.1 元 | **+0.2 元**（2×0.1） |
| 三级（文案重做） | LLM 全流程 +~0.15 元 | **+0.9 元**（7×0.1，仅分镜） |

若该段为可灵升级：二级重制 **+3 元**（2×1.5 std）；三级 **+10.5 元**（7×1.5）。

质量优先策略：**宁可多跑 Thinking 与 VL 二检**，静图重 1 张仅 **~0.1 元**，通过率优先。

## 7. 成本与质感开关

| 开关 | 效果 | 说明 |
| --- | --- | --- |
| `IMAGE_PROVIDER` | 切换文生图后端 | 默认 **`z_image_t2i`**；备选 `wan_t2i` |
| `Z_IMAGE_PROMPT_EXTEND` | 百炼自动扩写 Prompt | 默认 **false**（0.1 元/张）；开≈0.2 元/张 |
| `KLING_UPGRADE_ENABLED` | 指定段改动态视频 | 默认 **false**；可按 segment 开启 std |
| `HOST_ENABLED` | 片头/封面叠讲解人 | 资产就绪后开；无额外 API |
| 减少段数（6 段/条） | 出图约 **-12%** | 需改分镜逻辑 |
| 缩短单镜时长（8s/镜） | 出图约 **+160%** | 见 §3 分镜策略表 |
| 历史可灵 pro 方案 | 单条 ~22 元 | **已弃用为默认**，仅作对照 |

## 8. 各环节 API 用量（成本视角）

单条成片、一次通过、**7 分镜**基准（与 [需求.md §3](./需求.md#各环节-ai-一览按流水线顺序) 对照）：

| 环节 | 模型 | 单次成片约用量 | 备注 |
| --- | --- | --- | --- |
| 标题筛选 | V4-Flash ×2 | ~6k in / ~1.5k out | Non-thinking |
| 文案链路 | V4-Flash ×4 | ~20k in / ~10k out | 校验/质检用 Thinking |
| 画面 Prompt | V4-Flash ×1 | ~3k in / ~2k out | — |
| 分镜出图 | Z-Image（ImageProvider） | **7 张**（仅分镜） | 默认 ~0.1 元/张 |
| 动态升级（可选） | VideoProvider 可灵 std | 0～7 次视频 | 默认 **0** |
| 投稿封面 | FFmpeg / PIL | 从 `intro.png` 或成片首帧 | **0 元 API** |
| 片头 | FFmpeg 模板 | 1 条 MP4 | **0 元 API** |
| 配音字幕 | CosyVoice **`cosyvoice-v3-flash`** | ~1100 字 + 时间轴 | 音色 `longwan_v3`；按句合成，总字数计费 |
| 文本质检 | V4-Flash Thinking ×1 | ~5k in / ~2k out | 合并三轮为一次终审 |
| 画面质检 | Qwen-VL ×7 帧 | 7 分镜插画 | 可对可疑帧二检 |
| 音频质检 | 规则检测 | 1 条音轨 | 无 ASR |
| 回流 | V4-Flash（可选） | 日批 | 可忽略单条摊销 |

重制系数：二级 +1～3 张插画 +1～2 VL；三级 +全文案 LLM +分镜张数（7 或按策略）。若开启可灵升级，额外按段计费。API 预算按 **1.3 倍** Token、**1.25 倍** 出图预留。
