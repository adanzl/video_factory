# 提示词构建链路

> 审查日期：2026-07-21。范围：`backend/` 内 LLM / 出图相关提示词。  
> 风格：规则字符串硬编码 + `f-string` / `.format()` 拼装，无外部模板引擎。  
> 相关：`docs/LLM.md`（thinking 开关）、`docs/选题.md`（选题策略）。

---

## 总览

```text
job / 配置参数
  → build_*_prompts() 拼 system + user
  → llm_mgr / DeepSeekClient
  → 解析 JSON / 字符串
  → wrap / normalize / LoRA / 合规（可选）
  → 落库或调用出图 / I2V API
```

预览汇总入口：`script/compose.collect_prompts`（写入
`script["llm_prompts"]`）。

统一包装：`prompt_common.prompt_step` →
`{step, label, system, user}`。

## A. 标准脚本流水线

规则与拼装在：`backend/app/services/script/`。

| 路径 | 内容 |
| --- | --- |
| `prompt_common.py` | 共用辅助（profile、字数预算、`prompt_step`） |
| `voiceover_standard/` | 自由口播 A1/A4/A5；`styles/` 按风格拆规则 |
| `voiceover_standard/styles/common.py` | 跨风格共用规则与 JSON 样例 |
| `visual_brief.py` | visual_brief 规则与 A2 构建 |
| `image_prompt.py` | 六维规则、motion、A3、`wrap_image_prompts` |
| `segment_split.py` | 口播切分镜 `apply_segments_from_voiceover` |
| `voiceover_material/` | 素材口播 B1 构建 |

运行时：`llm_deepseek` → worker
`stages/standard/script.py`。

`visual_style` **不由 LLM 生成**，在
`llm_deepseek._resolve_visual_style` 按
`content_style` 硬编码。

### A1 口播文案

| 项 | 说明 |
| --- | --- |
| 入口 | `build_voiceover_standard_prompts` |
| 调用 | `DeepSeekClient._generate_narration` |
| 注入 | title、style、orientation、字数、补充信息、feedback |
| 产出 | `title`、`narration`、`word_count`（无 segments） |
| 后续 | `segment_split.apply_segments_from_voiceover` |

### A2 画面概述

| 项 | 说明 |
| --- | --- |
| 入口 | `build_visual_brief_prompts` |
| 调用 | `DeepSeekClient._fill_visual_briefs` |
| 注入 | 已切分 segments 的 text、全文 narration、visual_style、title |
| 产出 | 每段 `visual_brief`（不改 text） |
| 画风 | A2 前写入硬编码 style；缺省时 `_fill_visual_briefs` 按 job 补齐 |
| 约束 | 仅 `daily_story` 或已含 dialogue 时注入角色入画规则 |
| 字数 | `visual_brief` 统一 80–150 字（日常也走本入口） |

### A3 文生图 / 动效提示词

| 项 | 说明 |
| --- | --- |
| 入口 | `build_image_prompts` |
| 调用 | `fill_image_prompts`；API 亦可直接预览 |
| 规则选择 | `image_prompt_rule(orientation, content_style, sd15_mode)` |
| SD15 字段 | provider=`sd15_t2i` 时带 `sd15_prompt_en` |
| 分批 | `LLM_IMAGE_PROMPT_BATCH_SIZE` |
| 后处理 | `wrap_image_prompts`（如 daily_story 加参考图前缀） |
| 产出 | `image_prompt`、`motion_prompt`、可选 `sd15_prompt_en` |
| 落库 | `video_segment` 对应列 |

出图消费：`image_mgr.generate_segment_images`。

- SD15：优先 `sd15_prompt_en`
- 其它：`image_prompt`
- 含地图关键词时前置合规句

### A4 口播扩写

| 项 | 说明 |
| --- | --- |
| 入口 | `build_voiceover_standard_expand_prompts` |
| 时机 | 口播字数低于下限时 |
| mode | `narration_only`（标准）/ `material`（素材） |
| 风格 | `narration_only` 注入 `voice` / `anti_rep` |
| 上限 | 传入 `max_chars`（`accept_max`）防注水 |
| 调用 | `llm_deepseek._expand_narration_if_needed` |

### A5 单段缩句

| 项 | 说明 |
| --- | --- |
| 入口 | `build_voiceover_standard_shrink_prompts` |
| 时机 | 单镜口播过长 / 总口播略超上限 |
| cap | `segment_text_char_cap(..., chars_per_sec=job 语速)` |
| 调用 | `shrink_segment_texts`（贯通 `chars_per_sec`） |

## B. 素材管线

### B1 素材口播

- 入口：`build_voiceover_material_prompts`
- 时间表：`board_timeline.material_timeline_*`
  （表含对象 + 画面特征 + 字数范围）
- 写法：有表时注入三层 + 全片字数硬区间
- 预览：贯通 `speech_chars_per_sec`、`need_opening`
- 调用：`generate_material_script`；material job 的
  `collect_prompts`
- 产出：对齐视频时间轴的口播脚本

### B2 视频帧分析

- 入口：`VideoAnalyzer._build_prompts`
- 输入：抽帧 base64 + 间隔秒数
- 产出：时间表 JSON（title / segments）
- 校验：规范化起止与 duration；重叠裁剪；
  末段可贴齐总时长

## D. 日常故事

文件：`backend/app/services/daily_story/prompts.py`。  
流水线与待办总览：[日常故事.md](./日常故事.md)。

角色外貌常量 `DAILY_STORY_CHARACTERS` 同时进入
`visual_style` 与 A3 的 wrap 前缀。

### D1 主题生成

- 入口：`build_daily_story_theme_prompts(count)`
- 产出：主题列表

### D2 故事正文

- 入口：`build_daily_story_prompts(theme)`
- 产出：故事内容（对话剧本原料）

### D3 分镜脚本

- 入口：`build_daily_script_prompts(...)`
- 调用：daily_story stage / `job_mgr`
- 注入：`scene_title`、`setting`、对白、语速
  （`info.script.speech_chars_per_sec`；日常默认 3.6；
  旧任务顶层 cps 会 migrate 进 script）
- 产出：scenes（切镜 + dialogue + `shot_type`；**不**写画面概述）
- 时长：单镜上限与 stage 一致（18 秒；字数上限 = 18 × cps）
- 画面：随后统一走 A2 `fill_visual_briefs`（80–150 字）
- 目标镜数约 6–14；转折句特写置镜首，禁止碎镜

### D4 对话标题

- 入口：`optimize_title.build_chat_title_prompts`
- 调用：`worker/stages/daily_story/script.py`
- 产出：适合对话体的标题

## F. 出图侧二次加工

脚本阶段产出的字符串，在真正调 API 前可能再改一次。

### F1 SD15 英文化

| 项 | 说明 |
| --- | --- |
| 入口 | `build_sd15_prompt_system` / `build_sd15_prompt_user` |
| 调用 | `llm_mgr.prepare_sd15_image_prompt` ← `_prepare_sd15_prompt` |
| 时机 | 非「已是英文短 prompt」快路径时 |
| 产出 | 英文 `prompt_en`、business、可选左右分镜 |

### F2 SD15 最终串

| 项 | 说明 |
| --- | --- |
| 入口 | `build_sd15_full_prompt` |
| 组成 | LoRA 前缀 + subject + business 后缀 |
| 配置 | `SD_BUSINESS`、checkpoint / LoRA 解析 |

### F3 Agnes 出图校验

| 项 | 说明 |
| --- | --- |
| 入口 | `image_agnes._VERIFY_SYSTEM_PROMPT` |
| 用途 | 出图后 VL 清单质检（非生图 prompt） |
| 清单 | scene / zhao_hair / extra_arms / cast_count |
| 人数 | 主体 ≤3；昭昭/灿灿未发言也可同框 |
| 极性 | 各项「是」=通过侧；喂给 VL 的场景句会去掉 daily wrap 前缀 |
| 生图 | Agnes 原样提交 `prompt` 字段 |

### F4 I2V 动效

| 项 | 说明 |
| --- | --- |
| 默认句 | `_DEFAULT_MOTION_PROMPT`（wan / agnes 各一份） |
| 稳定化 | `_stabilize_motion_prompt`；空则填默认 |
| 服务端扩写 | `WAN_I2V_PROMPT_EXTEND`（本地不再拼模板） |

T2I 侧同类开关：`WAN_PROMPT_EXTEND`、
`Z_IMAGE_PROMPT_EXTEND`。

## 配置项（与提示词相关）

| 配置键 | 作用 |
| --- | --- |
| `LLM_IMAGE_PROMPT_BATCH_SIZE` | A3 分批大小 |
| `IMAGE_PROVIDER` | 是否生成 `sd15_prompt_en`、出图路径 |
| `SD_BUSINESS` | SD15 business 覆盖 |
| `WAN_PROMPT_EXTEND` | 万相 T2I 服务端扩写 |
| `Z_IMAGE_PROMPT_EXTEND` | Z-Image 服务端扩写 |
| `WAN_I2V_PROMPT_EXTEND` | 万相 I2V 服务端扩写 |
| `MAX_TITLE_LENGTH` | 标题字数进 A1/C/E1 |
| `TTS_INSTRUCTION` | TTS instruct（非 LLM） |
| `SKIP_SCRIPT_QUALITY_CHECK` | 是否跳过 image_prompt 质检 |

任务级覆盖：`info.content_style`、`orientation`、
`supplementary_info`、`image_provider`。

## 标准主路径时序

```text
A1 voiceover_standard
  → segment_split（切 segments）
  → A2 visual_brief
  → A3 image_prompt (+ wrap)
  → E1 标题优化（可选）
  → 出图：image_prompt 或 F1→F2
  → I2V：motion_prompt + F4
```

素材主路径：`B2`（可选入库分析）→
`B1 voiceover_material` → 后续视 pipeline 接出图。

日常故事：`D1` → `D2` → `D3`（切镜+对白）→
`A2 visual_brief` → `D4` 标题 → `A3 image_prompt`（+ wrap）。
